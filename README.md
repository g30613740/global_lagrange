# Global Lagrange Interpolation

Учебный проект по численным методам: **глобальная интерполяция функции полиномом Лагранжа**.

Программа на C++ генерирует случайные узлы на заданном отрезке, вычисляет значения функции, строит интерполяционный полином Лагранжа в глобальной форме и записывает результаты в файл. Python-скрипт загружает эти данные и строит график, сравнивая исходную функцию и полином. Требования преподавателя к задаче: не разрешается использовать std::vector вместо ручного управления памятью и практики ООП.

## Структура проекта

- `src/` – исходный код C++
- `include/` – заголовочные файлы
- `scripts/` – Python-скрипт для визуализации
- `Makefile` – сборка C++ части
- `requirements.txt` – зависимости Python

## Требования

- Компилятор C++ с поддержкой стандарта C++11 (например, `g++`)
- Утилита `make`
- Python 3.6 или выше
- Библиотеки Python: `matplotlib`, `numpy` (устанавливаются через `requirements.txt`)

## Быстрый старт

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/g30613740/global_lagrange.git
   cd global_lagrange
   ```

2. **Соберите C++ программу:**
   ```bash
   make
   ```

3. **Запустите программу – она создаст файл с данными parameters.txt:**
   ```bash
   ./a
   ```

4. **Установите зависимости Python (если ещё не установлены):**
   ```bash
   pip install -r requirements.txt
   ```

5. **Постройте график**
   ```bash
   python scripts/create_plot.py
   ```
В результате появится изображение lagrange_plot.png.

## Настройка параметров интерполяции

Вы можете изменять интерполируемую функцию, отрезок интерполяции, количество узлов и другие параметры, производя нужные изменения в файлах src/main.cpp или include/functions.h. После изменений пересобирайте программу (make) и запускайте заново.
